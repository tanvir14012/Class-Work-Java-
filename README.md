# Class-Work-Java-

- Bus: A ticket counter simulatior application. An admin puts entry for tours and a clerk allocates seat and passanger information using GUIs.

- Candy Crash: A console game for 2 player, each player tries to maximize their score by specifying appropriate position in a 8x8 grid.

- Calculator: A simple calculator application.

- Dbms: A GUI application that connects to Oracle Database and calls a stored procedure in the database to show some useful entry.

GUIs are developed using Java Swing.
